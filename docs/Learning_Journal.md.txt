## Day 1

### What is Medallion Architecture?

Bronze Layer:
Raw ingested data from source systems.

Silver Layer:
Cleaned, validated, and standardized data.

Gold Layer:
Business-ready tables used for analytics and reporting.


### Why Parquet?

Parquet is a column-based storage format.

Benefits:

- Smaller storage size
- Faster query performance
- Efficient for analytics workloads
- Industry standard for data lakes
